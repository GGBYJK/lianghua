# 头肩顶与反向头肩顶评分策略（信号触发即评分版）

> 适配信号规则：**右肩确认后，价格回落（反向：回升）到「右肩 ↔ 右颈」距离的一半时触发信号**。
> 因此本版本**取消所有以「颈线突破」为前提的得分**，改为以**信号触发时刻已经可观测的要素**评分，保证信号弹出时立即显示分数。
> 同时：① 用 **QTR 指标**衡量左右肩高度接近、左右颈价格接近；② 趋势背景仅评价**形态所在周期**的局部趋势（长期 H1/D1 趋势由策略既有趋势评分负责，不在此重复）。

---

## 一、评分目标

对**已识别且右肩已确认**的头肩顶 / 反向头肩顶形态进行量化评分。评分越高，代表结构越标准、右肩回落（回升）确认越干脆、交易价值越高。

- 总分 100 分制，结合封顶与惩罚规则避免「形似但无效」获得高分。
- 头肩顶判断上涨末端潜在反转；反向头肩顶判断下跌末端潜在反转。两者逻辑一致，只需将高低点、回落方向、目标方向反向处理。
- **评分时刻 = 信号触发时刻（颈线尚未突破）**，所有子项必须在该时刻可计算。

---

## 二、总分结构

| 评分维度 | 权重 | 核心判断（信号触发时刻可观测） |
|---|---|---|
| 趋势背景（形态所在周期） | 10 分 | 形态出现前，**当前周期**是否有明确趋势（头肩顶前上涨 / 反向前下跌） |
| 三峰 / 三谷结构 | 25 分 | 头部最突出，**左右肩高度接近（QTR 衡量）**，顺序清晰 |
| 颈线质量 | 15 分 | 颈线点明确，**左右颈价格接近（QTR 衡量）**，斜率合理，价格对颈线有尊重 |
| 时间对称性 | 10 分 | 左肩、头部、右肩之间的时间跨度不过度失衡 |
| 量能 / 动能配合 | 15 分 | 右肩动能衰减，右肩回落（回升）时放量，可结合 RSI/MACD 背离 |
| 回落确认（替代「突破确认」） | 20 分 | 右肩见顶（见底）结构明确，向颈线方向回落（回升）干脆、路径单向 |
| 交易价值 | 5 分 | 以**信号触发价**为入场参考的盈亏比是否合理 |

> 与原版差异：原「突破确认 20 分」依赖颈线突破，**信号触发时尚未发生**，已整体替换为**回落确认 20 分**（评价右肩反转质量与向颈线推进的质量）。

---

## 三、关键点定义

- **头肩顶**：左肩高点 LS、头部高点 H、右肩高点 RS、两个回调低点 N1/N2，N1–N2 连线为颈线。要求 `H > LS` 且 `H > RS`。
- **反向头肩顶**：左肩低点 LS、头部低点 H、右肩低点 RS、两个反弹高点 N1/N2，N1–N2 连线为颈线。要求 `H < LS` 且 `H < RS`。
- **右颈价 `neckline_at_RS`**：颈线在右肩位置（按 N1–N2 斜率外推到 RS 时间）的价格。
- **信号触发价 `P_signal`**：
  - 头肩顶：`P_signal = RS − 0.5 × (RS − neckline_at_RS)`
  - 反向头肩顶：`P_signal = RS + 0.5 × (neckline_at_RS − RS)`
- **QTR**：用于度量价格距离的波动单位（你已引入的指标）。下文所有「接近度」均以 **QTR 的倍数**衡量，倍数阈值为建议初始值，需回测校准。

---

## 四、详细评分规则

### 1. 趋势背景（形态所在周期）— 10 分

> 只评价**形态出现的这个周期**的局部趋势（即从形态左侧、LS 之前的走势到 H 的方向）。长期跨周期趋势由策略既有趋势评分负责，此处不重复。

| 子项 | 分数 |
|---|---|
| 形态前在**当前周期**存在明确趋势（头肩顶前上涨 / 反向前下跌） | 5 |
| 该趋势在当前周期持续足够长，不是横盘区间里强行凑形态 | 3 |
| 形态靠近当前周期的重要压力 / 支撑、前高 / 前低或关键均线 | 2 |

- 当前周期无明确前置趋势 → 本项 ≤ 2 分，并触发封顶规则（见第五节）。

### 2. 三峰 / 三谷结构 — 25 分

| 子项 | 分数 |
|---|---|
| 头部明显高于左右肩（反向：明显低于左右肩） | 8 |
| **左右肩高度接近（QTR 衡量，见下表）** | 6 |
| 左肩、头部、右肩的结构顺序清晰 | 5 |
| 右肩未突破头部（反向：右肩未跌破头部） | 4 |
| 中间没有过多破坏结构的杂峰 / 杂谷 | 2 |

**左右肩高度接近（QTR 评分，满分 6）：**

```
shoulder_diff = abs(LS − RS)          # 价格差
ratio_S       = shoulder_diff / QTR   # 以 QTR 为单位
```

| `ratio_S` | 得分 |
|---|---|
| ≤ 0.5 × QTR | 6 分 |
| 0.5 – 1.0 × QTR | 4 分 |
| 1.0 – 1.5 × QTR | 2 分 |
| > 1.5 × QTR | 0 分（并参与惩罚） |

> 取代原「肩部相似度 = 1 − abs(LS−RS)/pattern_height」。用 QTR 而非形态高度做分母，使阈值在不同波动率品种间更稳定。

### 3. 颈线质量 — 15 分

| 子项 | 分数 |
|---|---|
| N1、N2 两个颈线点清晰 | 5 |
| **左右颈价格接近 / 斜率合理（QTR 衡量，见下表）** | 4 |
| 信号触发前价格多次尊重颈线 | 3 |
| 颈线附近没有大量假突破 / 无效刺破 | 3 |

**左右颈价格接近（QTR 评分，满分 4）：**

```
neck_diff = abs(N1 − N2)          # 两个颈线点的价格差
ratio_N   = neck_diff / QTR
```

| `ratio_N` | 得分 |
|---|---|
| ≤ 0.5 × QTR | 4 分 |
| 0.5 – 1.0 × QTR | 3 分 |
| 1.0 – 1.5 × QTR | 1 分 |
| > 1.5 × QTR | 0 分（并参与惩罚） |

> `neck_diff` 越小，颈线越水平、形态越标准。颈线可轻微上斜 / 下斜，但 `ratio_N > 1.5` 视为倾斜过大。

### 4. 时间对称性 — 10 分

```
左侧时间 = H_time − LS_time
右侧时间 = RS_time − H_time
时间比例 = max(左侧时间, 右侧时间) / min(左侧时间, 右侧时间)
```

| 时间比例 | 得分 |
|---|---|
| 1.0 – 1.8 | 8 – 10 分 |
| 1.8 – 3.0 | 4 – 7 分 |
| > 3.0 | 0 – 3 分 |

> 一侧极短、一侧极长，通常说明形态不标准，降低结构可信度。

### 5. 量能 / 动能配合 — 15 分

> 全部基于信号触发时刻已形成的 K 线，不依赖颈线突破时的放量。

| 子项 | 分数 |
|---|---|
| 从左肩到头部再到右肩，动能逐步衰减 | 5 |
| 右肩成交量低于左肩或头部 | 4 |
| **右肩见顶（见底）后回落（回升）时成交量放大 / 动能放大** | 4 |
| RSI / MACD 在右肩出现背离确认 | 2 |

> 原「突破颈线时成交量放大」依赖突破，已替换为「右肩回落 / 回升时放量」——该放量在信号触发时刻已可观测（价格已走完右肩到右颈的一半）。
> 若品种成交量不可靠（部分外汇、指数等），改为动能评分：RSI 背离 5 分、MACD 背离 4 分、波动率收缩 3 分、右肩回落动能 3 分。

### 6. 回落确认（替代「突破确认」）— 20 分

> 信号在「右肩 ↔ 右颈」中点触发，此时颈线尚未突破。本项评价**右肩反转质量**与**向颈线推进的质量**，是即时确认的核心。

| 子项 | 分数 |
|---|---|
| 右肩见顶（反向：见底）结构明确：顶 / 底分型、明显反转 K 线、长上影 / 长下影 | 6 |
| 从右肩到信号点的回落（回升）干脆，斜率 / 动能强，不拖泥带水 | 5 |
| 回落（回升）路径单向：途中未重新逼近 / 突破右肩高点（反向：未跌破右肩低点） | 4 |
| 回落（回升）过程伴随放量或动能放大（卖压 / 买压释放） | 3 |
| 信号点已明确脱离右肩极值区、向颈线方向有顺畅运行空间（颈线附近无密集反复支撑 / 阻力） | 2 |

> 头肩顶以「右肩见顶 + 向下回落过半」作为即时确认；反向头肩顶以「右肩见底 + 向上回升过半」作为即时确认。
> 注意：颈线**是否最终突破**不在本评分内——评分服务于「信号弹出即显示」，突破与否由后续行情验证。

### 7. 交易价值 — 5 分

> 以**信号触发价**为入场参考（比等颈线突破入场更早，盈亏比通常更优）。

```
# 头肩顶
目标价 = neckline_at_RS − (H − neckline_at_RS)
止损价 = RS 上方（可加 缓冲，如 RS + k×QTR）
盈亏比 = (P_signal − 目标价) / (止损价 − P_signal)

# 反向头肩顶
目标价 = neckline_at_RS + (neckline_at_RS − H)
止损价 = RS 下方（可减 缓冲，如 RS − k×QTR）
盈亏比 = (目标价 − P_signal) / (P_signal − 止损价)
```

| 盈亏比 | 得分 |
|---|---|
| ≥ 2.0 | 5 分 |
| 1.5 – 2.0 | 3 分 |
| 1.2 – 1.5 | 1 分 |
| < 1.2 | 0 分 |

> 目标价仍按经典测幅（头部到颈线的距离）计算，只是以颈线价为投影基准，无需等待突破即可给出。

## 五、封顶与惩罚规则

| 条件 | 处理 |
|---|---|
| 当前周期没有明确前置趋势 | 总分封顶 **70** 分 |
| 头部不明显高于 / 低于左右肩 | 总分封顶 **50** 分 |
| 左右肩高度差 `ratio_S > 1.5 × QTR`（严重不对称） | 总分额外扣 **10 – 20** 分 |
| 左右颈价格差 `ratio_N > 1.5 × QTR`（颈线倾斜过大） | 总分额外扣 **5 – 15** 分 |
| 右肩回落 / 回升动能不足，或途中重新逼近 / 突破右肩极值（路径被破坏） | 总分封顶 **60** 分 |
| 颈线附近多次假突破 / 假刺破 | 总分额外扣 **5 – 15** 分 |

> 已删除原版「没有有效突破颈线：封顶 75」「突破后快速回到颈线内部：封顶 60」——本策略在突破前发信号，这两条不再适用，相应风险由「回落确认」与「路径被破坏」封顶承接。

---

## 六、最终评级

| 分数区间 | 评级 | 解释（信号触发时刻 / 颈线突破前） |
|---|---|---|
| 85 – 100 | A 级 | 结构标准、右肩回落确认充分，信号质量高 |
| 70 – 84 | B 级 | 形态基本有效，存在部分瑕疵 |
| 55 – 69 | C 级 | 疑似形态，需结合后续走势进一步确认 |
| 40 – 54 | D 级 | 形似但质量差，参考价值较低 |
| < 40 | 忽略 | 不建议作为头肩形态处理 |

## 七、程序化评分流程建议

```
# 信号触发瞬间执行，立即返回分数
final_score =
      trend_score          # 形态所在周期的局部趋势，10
    + structure_score      # 三峰/三谷，含 QTR 肩高接近，25
    + neckline_score       # 颈线质量，含 QTR 左右颈接近，15
    + time_score           # 时间对称性，10
    + momentum_score       # 量能/动能，15
    + pullback_score       # 回落确认（替代突破确认），20
    + trade_value_score    # 以信号触发价计的盈亏比，5

final_score = apply_caps_and_penalties(final_score)
```

- 所有子项均在**右肩 ↔ 右颈中点触发的瞬间**可计算，信号弹出即显示评分。
- 80 分以上进入重点观察池；70 – 80 分作为辅助信号；低于 70 分仅做记录。

## 八、实操建议

- 信号已在突破前发出，因此评分核心从「突破确认」转移到「右肩反转质量 + 回落动能」。
- QTR 的倍数阈值（0.5 / 1.0 / 1.5）为建议初始值，请用回测校准；不同品种、不同周期可分别校准。
- 趋势背景只评当前周期，避免与策略既有的 H1/D1 长期趋势评分重复计分。
- 不同周期可分别评分，大周期高分形态优先级更高。
